# 🐝 Kalender Generator Bienen

Ein einfacher webbasierter Generator für imkerliche Zeitpläne.

## 📌 Funktion

Dieses Tool erstellt automatisch Zeitpläne für verschiedene imkerliche Arbeitsabläufe 
(z. B. Zucht, TuB, Königinnen sperren).  
Aus einem Startdatum werden alle relevanten Arbeitsschritte berechnet und als Liste angezeigt.

## ⚙️ Funktionen

- Auswahl verschiedener Zeitpläne
- Automatische Berechnung aller Termine
- Export/Download als:
  - 📅 ICS-Kalenderdatei (Google Kalender, Samsung Kalender etc.)
  - 📄 PDF-Übersicht
- Läuft komplett im Browser (keine Installation nötig)

## 🌐 Nutzung

Die Nutzung erfolgt ohne Gewähr für Schäden irgendwelcher Art.
